## Run

```
pip install -r requirements.txt
fastapi dev main.py
```

## Virtual environment

Install deps:

```
pip install -r requirements.txt
```

Create .venv:
```
python -m venv .venv
```

Activate:

```
source .venv/bin/activate
```

Deactivate:

```
deactivate
```

Generate requirements.txt:
```
pip freeze > requirements.txt
```
